# Grafik-Assets

Die verbindlichen Regeln stehen in [`ART_BIBLE.md`](../ART_BIBLE.md).

- `source/`: editierbare, von Hand erstellte SVG- und Texturquellen
- `runtime/`: geprüfte, optimierte Laufzeitdateien

Neue Assets werden erst nach dem Review aus der Quelle in den Laufzeitordner übernommen. Leere Fachordner werden bei der ersten zugehörigen Grafik angelegt; dadurch bleiben keine scheinbar fertigen, aber inhaltslosen Assetbereiche bestehen.
